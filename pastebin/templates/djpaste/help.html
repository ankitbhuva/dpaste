DjPaste is a pastebin written in Django. It was built as part of My MCA 5th Semester mini-project. It is heavily stolen inspired from Dpaste.com, but the codebase is in no way related.

Pasting content with DjPaste is very easy. You just choose the type of the code you are pasing, paste, and hit the button. Magic starts happening. Only text is the required field. All other fields are optional.

Super Special Features:

  • Automatically focuses to Code field, so you can paste fast.
  • Remembers previous choices.
  • No registration needed. just open and get the work done.
  • No unnecessary fields.
  • You can find code at github link. it's free and open to use so do whatever you wanna do with it. :p

Coming soon features :

  • Adding blog archives page. it will enable users to fastly navigate through and search blog posts with arranged dates.
